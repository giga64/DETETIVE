## 🌐 Domínio Personalizado

Para usar um domínio personalizado:

1. Compre um domínio
2. Configure DNS apontando para `seu-usuario.github.io`
3. Edite o arquivo `CNAME` com seu domínio
4. Configure no GitHub: Settings > Pages > Custom domain
